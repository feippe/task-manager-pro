## Overview

Task Manager Pro is a simple but fully functional **web application** that allows users to create, complete, delete, and prioritize tasks.  
The purpose of this project was to learn and apply core web development concepts using **HTML, CSS, and JavaScript**, as well as to generate dynamic content through user interaction and browser-local storage.

The app includes:
- Two HTML pages (`index.html` and `report.html`)
- A dynamically generated report using JavaScript
- Fully interactive user input and DOM updates
- Persistent data using `localStorage`
